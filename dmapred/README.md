# Dmapred

**Distributed MapReduce Systems in Elixir**

## Goal
Demos of distributed system
## Running
```
mix deps.get && mix compile
```
Master and all workers should be run on different terminals/windows.
### Starting Master
``` make start_master ```

### Starting Workers
``` make start_worker1 ```
``` make start_worker2 ```
``` make start_worker3 ```
